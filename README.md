## Setup

- Bar: `dmenu.`
- Compositor: `picom.`
- Shell: `starship.`
- Terminal Emulator: `st.`
- Text Editor: `nVim.`
- MPD client: `ncmpcpp.`
- Web Browser: `cutebrowser.`
- Window Manager: `Suckless DWM.`
- Theme: `Catppuccin.`

## Installation

- Download repo: clone --depth=1.
- Inspect dotfiles.
- Find the stuff you like.
- Change, add to your dotfiles.
